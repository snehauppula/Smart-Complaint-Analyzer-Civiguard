# 🚨 Smart Complaint Analyzer - CIVIGUARD

An intelligent complaint analysis system that processes and categorizes user complaints using Machine Learning and Data Science techniques. This project aims to assist authorities in efficiently handling public grievances by automating complaint classification and insights generation.

---

## 📖 Overview

CIVIGUARD is a data-driven platform designed to:
- Analyze complaint data
- Categorize complaints automatically
- Provide insights for better decision-making

It reduces manual effort and helps in prioritizing critical issues.

---

## ✨ Features

- 📊 Complaint Data Analysis
- 🤖 Machine Learning-based Classification
- 🧹 Data Preprocessing (Cleaning, Handling Missing Values)
- 📈 Visualization of Complaint Trends
- 🌐 Web Interface using Flask
- 📁 CSV Upload Support
